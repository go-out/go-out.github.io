住吉踊 🌾 Sumiyoshi Odori

神功皇后が堺の浜に上陸した際、住民たちが祝福と歓迎のために踊った舞に由来するとされる「住吉踊」
六月十四日の御田植神事のほか、住吉祭・観月祭・お正月期間など、一年を通して奉仕される住吉大社の名物踊り

神へ豊作を願う音頭に合わせて、童女たちが「心」の字を書くように飛び跳ね、団扇を打ちながら踊る
