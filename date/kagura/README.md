日本神話（古事記・日本書紀）の一節「天岩戸隠れ」に登場する神様の舞が起源とされる、日本を代表する歌舞・神事芸能

一般的に、宮中で宮内庁式部職楽部（くないちょうしきぶしょくがくぶ）が執り行う神楽「御神楽（みかぐら）」と、民間で行われるそれ以外の神楽「里神楽（さとかぐら）」に大別される
（里神楽を持つ地域は全国的に珍しく、「伊勢神楽」「出雲神楽」・九州の「高千穂神楽」・関西地方の「浪速神楽」などが代表的）
