関西地方（京阪神）を中心に伝わる里神楽「浪速神楽（なにわかぐら）」
鈴・扇・榊（さかき）などの神聖な採物（とりもの）を持った巫女が舞う「巫女神楽（神懸かり系・早乙女系）」の系統
（現在の浪速神楽は全て巫子だけで舞われるが、江戸時代には神主も舞っていた）

現在大阪府内で最も普及している浪速神楽は、故冨永正千代楽師を家元とする流派「冨永流浪速神楽（とみながりゅうなにわかぐら）」
大正九年頃に冨永正千代が発足した「関西雅楽松風会（かんさいががくしょうふうかい）」が、江戸時代から伝承が途絶えてほぼ滅びた浪速神楽を復興・完成させた

松風会の会長（2024年現在）は、大阪府富田林市にある佐備神社で稽古・奉納を行うほか、大阪天満宮・住吉大社などでも神楽を奉仕し、二十八座の冨永流浪速神楽を継承

装束は、白衣に緋（濃く明るい赤色）の袴・平絹の舞衣（まいぎぬ）
その上に、金襴の「千早（ちはや）」と呼ぶ衣裳を着る

演奏に用いる楽器は主に四種「龍笛（りゅうてき・横笛）」「楽太鼓（がくだいこ）」「手拍子（ちゃんぽん）」「笏拍子（しゃくびょうし）」
