体を崩しやすい春から夏への季節の変わり目に健康を祈願する、古代の国家祭祀「鎮花祭」を起源とする祭事

日本神話の時代から継承される、毎年四月十八日に大神神社（狭井神社）で行われる「鎮花祭（はなしずめまつり）」のほか、全国の神社でも同様の祭りが行われる
