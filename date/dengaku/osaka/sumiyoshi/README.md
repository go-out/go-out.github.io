御田植神事 Otaue Shinji (Sacred Rice Planting)

場所：住吉大社 ⛩️ Sumiyoshi Taisha
実施日：毎年 六月十四日 午後一時から
July 14th every year start at 1pm

楽人・八乙女・御稔女・植女・替植女・稚児・風流武者・住吉踊りの踊子など、たくさんの参加者が芸能を披露する住吉大社に伝わる田楽（御田植祭）
境内の御田（おんだ）で田植えを行うと同時に、神楽女（巫女）による「田舞」・地元に伝わる「田植歌・田植踊」、鎧兜をまとった武士による風流武者行事などを披露
最後は、約150名の女児によるこの神事を起源とする住吉大社の名物踊り「住吉踊」で締めくくられる

※春から初夏にかけての御田には、蓮華が植えられている

宝之市神事 🌾 Takatra no Ichi Shinji

住吉大社 Sumiyoshi Taisha
毎年 十月十七日 午前十時
October 17th every year start at 10am

御田で育った稲穂を刈り取り、五穀と共に神前に奉献する特殊神事
境内では、商都大阪の起源とも伝えられている日本最初の市「升の市（ますのいち）」が催される
