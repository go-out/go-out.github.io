「神楽（日本の神道において神に奉納する歌舞）」には、宮中（皇居）で行われる「御神楽（みかぐら）」と、民間で行われる「里神楽（さとかぐら）」がある
（里神楽を持つ地域は全国的に珍しく、「伊勢神楽」「出雲神楽」・九州の「高千穂神楽」・関東の「江戸神楽」などが代表的）

関西地方（京阪神）を中心に伝わる里神楽「浪速神楽（なにわかぐら）」は、鈴・扇・榊（さかき）などの神聖な採物（とりもの）を持って巫女が舞う「巫女神楽（神懸かり系・早乙女系）」の系統

現在大阪府内で最も普及している浪速神楽は、故冨永正千代楽師を家元とする流派「冨永流浪速神楽（とみながりゅうなにわかぐら）」
大正九年頃に冨永正千代が発足した「関西雅楽松風会（かんさいががくしょうふうかい）」が、江戸時代から伝承が途絶えてほぼ滅びた浪速神楽を復興・完成させ、二十八座の冨永流浪速神楽を継承している
