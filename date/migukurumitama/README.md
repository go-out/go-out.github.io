一ノ宮参拝道 🇯🇵 眞名井ヶ原
美具久留御魂神社 ⛩️ Migukurumitama Jinja

「眞名井ヶ原」と称する山そのものを御神体とする自然そのままの緑豊かなお宮
眞名井ヶ原を含む境内全域は、大阪みどりの百選また大阪府自然環境保全地域に指定されている

本殿を囲む形で設けられた御神体山頂への山道「一ノ宮参拝道」には、古代の行政区分畿内（山城・大和・河内・和泉・摂津）七道（西海道・山陽道・山陰道・南海道・東海道・東山道・北陸道）の一ノ宮をお祀りする八本の御神木があり、御神体山参道を一周することで全国の一ノ宮参拝が出来る
