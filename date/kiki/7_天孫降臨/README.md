# 天孫降臨
アマテラスは、孫の「瓊瓊杵尊（ニニギ）」に葦原の中つ国の統治を委任した
ニニギはアマテラスから譲り受けた「三種の神器」を持ち、五柱の神々「五伴緒（いつとものお）」などを引き連れ、猿田毘古（サルタヒコ）[^1]に道案内を頼み、筑紫の東方の尊い峰「高千穗（たかちほ）」に降臨し、宮殿を建てた
[^1]:伊勢国五十鈴川のほとりに鎮座したとされ、天照大神以前に伊勢で信仰されていた太陽神だったとする説もある

## 五伴緒（いつとものお）
天児屋命（あめのこやねのみこと） - 神事・祭祀をつかさどった氏族、中臣氏及び藤原氏の祖神
布刀玉命（ふとたまのみこと） - 祭具作製・宮殿造営を担った氏族、忌部氏のち斎部氏（いんべうじ）の祖神
天宇受賣命（あめのうずめのみこと） - 巫女として祭祀に携わっていた氏族、猿女君の祖神
伊斯許理度売命（いしこりどめのみこと） - 八咫鏡（やたのかがみ）を作った神
玉祖命（たまのおや） - 八尺瓊勾玉（やさかにのまがたま）を作った神

ニニギ・五伴緒を護衛した二柱の神は、古代日本における軍事氏族の祖神
* 天忍日命（あめのおしひのみこと） - 摂津国住吉郡（現大阪市住吉区帝塚山）を拠点とした大伴氏の祖神
* 天津久米命（あまつくめのみこと） - 現存する日本最古の国風歌舞（くにぶりのうたまい）「久米舞（くめまい）」に関係する久米氏の祖神

ニニギは笠沙の岬で、美しい娘「木花之佐久夜毘売（このはなのさくやびめ）[^2]」と出会い、とても醜い姉の「石長比売（いわながひめ）[^3]」とともに嫁入りすることを姉妹の父「大山積神（おおやまづみのかみ）」[^4]に約束するが、ニニギは醜い妹を追い払い、木花之佐久夜毘売との間に三柱の神を得る
