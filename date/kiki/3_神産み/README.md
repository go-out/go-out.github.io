# 神産み（かみうみ）
国産みという大事を終えたイザナギとイザナミは、威力ある男神「大事忍男神（おおごとおしおのかみ／おほごとおしをのかみ）を得る
そして、地にたくさんの神々（国津神）を生み出す更なる大事「神産み」が始まる

***

イザナギとイザナミは、家宅を司る神々「家宅六神（かたくろくしん）」や、自然（海・水・風・木など）にまつわる神々など、16柱の子を得るが、火の神「火之迦具土神（カグツチ）」を産む際に女陰が焼けたイザナミは病気になった

イザナミは病に苦しみ、吐瀉物・大便・尿から次々と神が生まれた後、ついに死んだ[^1]
そして、イザナミの死を悲しんだイザナギの涙、怒ったイザナギが切り殺したカグツチの血、殺された火之迦具土神の体からも、たくさんの神々が産まれた
[^1]:イザナミは出雲国と伯伎国（伯耆国）の境にある比婆（ひば）の山（現在の島根県安来市域内）、または、紀伊国の熊野の有馬村（花窟神社・産田神社が比定地）に葬られたとされている

イザナギはイザナミを取り戻そうと黄泉国へ行くも、イザナミの体は腐って蛆がたかり、蛇の姿をした八雷神がまとわりついていた
イザナミが黄泉の住者になってしまったことを思い知ったイザナギは、黄泉との境である「黄泉比良坂（よもつひらさか）[^2]」を大岩で塞ぎ、永遠に離別することとなった
[^2]:島根県松江市東出雲町揖屋（揖夜神社付近）の山道「伊賦夜坂（いふやのさか）」が黄泉比良坂の比定地とされている

## 禊（みそぎ）
イザナギは黄泉の穢れから身を清めるため、竺紫（つくし）の日向（ひむか）の橘の小門（をど）の「阿波岐原（あはきはら）[^3]」で、禊（みそぎ）を行った
[^3]:現在の宮崎県宮崎市阿波岐原町、日向灘に面する松林およびシーガイア関連施設が置かれる海岸線が比定地

衣を脱ぐと十二柱の神々が生まれ、海に潜って身を清めたときに二柱の神、その禍（まが）を直そうとすると三柱の神が生まれた

続いて、水の底・中程・表面で身を清めると、六柱の海の神々が生まれた
* 綿津見三神（わたつみさんじん） - 海の守護神 全国の綿津見神社、海神社の祭神（娘には豊玉姫と玉依姫や、豊玉姫の子には阿曇磯良がいる・阿曇氏の祖神）
* 住吉三神（すみよしさんじん） - 日本全国に約600社ある住吉神社の祭神（底筒之男神（そこつつのおのかみ）・中筒之男神（なかつつのおのかみ）・上筒之男神（うわつつのおのかみ）の総称）

## 三貴子（みはしらのうずのみこ）
最後に左目を洗うと天照大御神、右目を洗うと月読命、鼻を洗うと建速須佐之男命が生まれた

* 天照大御神（あまてらすおほみかみ）- 伊勢神宮内宮（神宮 皇大神宮）・全国各地にある「神明神社」などに祀られている
* 月読命（つくよみのみこと）- 伊勢神宮内宮（神宮 皇大神宮）の別宮「月讀宮」・伊勢神宮外宮（豊受大神宮 別宮）の「月夜見宮」などに祀られている
* 建速須佐之男命（たけはやすさのをのみこと） - 熊野大社（島根県松江市）などの祭神（神仏習合では牛頭天王と同一視され、全国の祇園社・天王社、八坂神社にも祀られている）

イザナギは三柱の貴い子を得たと喜び、アマテラスに日（天）・ツクヨミに夜（暦）・スサノオに海原（地）を託したが、スサノオはそれを拒み、「母（イザナミ）に会いたい」と長い間泣き喚いた

呆れて怒ったイザナギはスサノオを追放し、最初に生んだ淡路島の「多賀の地の幽宮（かくりのみや、終焉の御住居）[^4]」に篭った
[^4]:現在の伊弉諾神宮（兵庫県淡路市多賀）が比定地
