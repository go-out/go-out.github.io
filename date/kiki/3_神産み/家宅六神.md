# 家宅六神
家宅を表す（または守る）六柱の神

1. 石土毘古神（いはつちびこのかみ） - 土を司る男神
2. 石巣比売神（いはすひめのかみ） - 砂を司る女神

3. 大戸日別神（おほとひわけのかみ） - 門の神の一つ

4. 天之吹男神（あめのふきおのかみ）- 屋上を司る男神
『古事記伝』では、「大祓詞」に登場する気吹戸主（いぶきどぬし）と同神と解釈している

5. 大屋毘古神（おおやびこのかみ） - 災厄の神霊
イザナギの禊の際に生まれる、大禍津日神と同神とされる

6. 風木津別之忍男神（かざもつわけのおしおのかみ） - 家屋の耐久性の表象
『古事記伝』では、底筒男神、または大祓詞の速佐須良比売と同神としている
