# 葦原中津国平定（国譲り）
アマテラスは、大変騒がしく手に負えない「葦原の中つ国（あしはらのなかつくに）」（天と地の「地」）を自分の子孫が収めるべきだと考え、知恵の神「常世思金神[^1]」と八百万の神が作を練り、正勝吾勝々速日天忍穂耳命[^2]・天菩比神[^3]・天若日子[^4]を地に派遣したが、遣いに出た神々はスサノオの子孫で出雲国を収めていた「大国主（おおくにぬし）」に支えてしまい、統治は進まなかった

[^1]:オモイカネは、信濃国（長野県・岐阜県中津川市の一部）の国造りの祖となり、阿智神社・戸隠神社などに祀られる
[^2]:アメノオシホミミは、のちに葦原の中つ国を統治する「瓊瓊杵尊（ニニギ）」の父
[^3]:アメノホヒは、古墳時代に古墳造営や葬送儀礼に関った氏族、土師氏（はじうじ、はじし）の祖神
[^4]:アメノワカヒコは、大国主神の娘「下照比売（シタテルヒメ）」との恋に溺れて使命を放棄する悲劇的かつ反逆的な神（「アマノジャク」の元となったとする説がある）

最終的に、大国主とその子（事代主（ことしろぬ）[^5]・建御名方（たけみなかた）[^6]）は、「建御雷神（たけみかづちのお）[^7]」に、「出雲大社（富足る天の御巣の如き）」の修復を条件に降参し、アマテラス（高天原）に国を譲り渡した

[^5]:明治以降、伊豆諸島の島産み・開拓の神として「三嶋大社」などに祀られる
[^6]:諏訪大社（長野県諏訪市ほか）の祭神として祀られていることから「諏訪神」とも呼ばれる
[^7]:鹿島神宮（茨城県鹿嶋市ほか）の祭神として祀られていることから「鹿島神」とも呼ばれる
