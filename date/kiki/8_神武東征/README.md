# 神武東征（じんむとうせい）
鸕鶿草葺不合尊と玉依姫との間に産まれた四柱の神は、葦原の中つ国を治めるため、東にある青々とした山に囲まれた美しい国「大和」へ東征[^1]に出た

* 五瀬命（イツセ）
* 稲氷命（イナヒ）
* 御毛沼命（ミケヌ）
* 神日本磐余彦尊（カムヤマトイワレビコ）

[^1]:道中、足一騰宮（妻垣神社・妻垣山）・岡田宮（福岡県北九州市）・多祁理宮（広島県安芸郡府中町）・吉備高島宮（岡山市南区宮浦にある高島が最も有力とされている）に一時的な宮殿「行宮（あんぐう/かりみや）」を建てた

***

浪速国へ着くと、大和の指導者の一人「長髄彦（ながすねひこ）」の軍が待ち構えていた
孔舎衛坂[^2]で合戦となり、長髄彦が放った矢は「五瀬命（イツセ）」を死に至らせる[^3]
[^2]:大阪府と奈良県との境、生駒山地を越える坂とされる
[^3]:和泉の國の「茅渟の海（現在の大阪湾）」で血を洗い流し、紀伊の國の竈山（竈山神社）で死んだ

神日本磐余彦尊は熊野で、尾張国造の祖「高倉下（たかくらじ）[^4]」に出会い、霊剣「布都御魂（ふつのみたま）[^5]」を譲り受ける
[^4]:神倉神社及び熊野速玉大社の神倉宮などに祀られている
[^5]:この剣（佐士布都神・甕布都神ともいう）は、石上神宮に祀られている

その後、八咫烏の案内で、熊野から吉野の川辺を経て、大和の宇陀（奈良県東部、宇陀市および宇陀郡周辺を指す地域名称）へ行き「兄宇迦斯・弟宇迦斯」と戦い、忍坂の大室（菟田の高倉山）で「土雲の八十建」「登美毘古（ナガスネビコ）」「兄師木（エシキ）・弟師木（オトシキ）」を一斉に打ち殺し、最後に国見丘で「長髄彦（ながすねひこ）」を撃ち、大和に勝利し「久米歌」を歌った

そこに邇藝速日命（ニギハヤヒ）が参上し、亂暴な神たちや多くの土雲（豪族）を服従させたとし、紀元前660年（神武天皇元年）「橿原の宮」で、イワレビコが日本国の初代天皇「神武天皇」として即位した

その後、三輪の大物主神の子である「比売多多良伊須気余理比売（ヒメタタライスケヨリヒメ）」を皇后とし、神武天皇は三柱の子を得た
