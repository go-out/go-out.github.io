# 誓約（うけい）
母に会うため「根の国（ねのくに）」へ向かう前に、スサノオはアマテラスに清明を誓うため、誓約・宇気比（うけひ）を行う

アマテラスは、スサノオの十拳劔（とつかのつるぎ）を噛み砕き、宗像三女神（むなかたさんじょしん）を産む
1. 多紀理毘売命（タキリビメ） - 宗像大社奥津宮の祭神
2. 多岐都比売命（タギツヒメ） - 宗像大社中津宮の祭神
3. 市寸島比売命（いちきしまひめのみこと）） - 宗像大社辺津宮の祭神
（女神らは「道主貴（みちぬしのむち）」とも呼ばれ、あらゆる「道」の最高神として崇敬を集めている）

次いで、スサノオはアマテラスの「八尺の勾玉の五百箇のみすまるの珠」を噛み砕き、五柱の男神を産む
1. 正勝吾勝勝速日天之忍穂耳命（まさかつあかつかちはやひあめのおしほみみのみこと）- 天皇・皇室の祖
2. 天之菩卑能命（あめのほひのみこと）- 出雲氏等の祖
3. 天津日子根命（あまつひこねのみこと）- 多くの氏族・国造の祖
4. 活津日子根命（いくつひこねのみこと）- 生國魂神社の祭神と同神
5. 熊野久須毘命（くまのくすびのみこと）- 熊野氏族（熊野三山）の祖神

（宗像三女神はスサノオの子・五柱の男神はアマテラスの子となる）

***

宗像三女神を得たスサノオはアマテラスに勝利を宣言し、アマテラスの持ち物や家を壊し、食事に糞をかけるなどの乱暴を働いた
アマテラスはスサノオを恐れ怒り、「天岩戸（あまのいわと）[^1]」に引き篭ってしまう
[^1]:天岩戸は天（高天原）にあるとされるが、天岩戸神社（宮崎県）・元伊勢内宮（京都府福知山市）など、全国各地に天岩戸と呼ばれる場所が多数存在する

## 天の岩戸（あまのいわと）

アマテラスが天岩戸に隠れたあと、天地は全く暗くなり夜がずっと続き、さまざまな禍（まが）が発生したので、無限の神々「八百万の神」がそこに集まり、知恵の神「常世思金神」の考えのもと、長鳴鶏（ながなきとり）を集めて鳴かせ朝を伝え、三種の神器のうち「八咫鏡（やたのかがみ）[^2]」と「八尺瓊勾玉（やさかにのまがたま）[^3]」を作らせ、天の香山（奈良県橿原市に同名の山がある）の榊（サカキ）などを飾り[^4]、祝詞（のりと）[^5]に合わせて、笑いと狂気の舞を激しく舞った[^6]
それを見た八百万の神はどっと笑い、不思議に思ったアマテラスはついに天の岩戸から顔を出すと、天地に自然と日が照った[^7]

[^2]:伊勢神宮にある御神体と、皇居にある御神体をかたどって作った形代の二つがある
[^3]:皇居にある御所の「剣璽の間」に、剣（形代）とともに保管されているという勾玉
[^4]:玉串や注連縄のルーツとなった神「布刀玉命（ふとたまのみこと）」が榊を飾った
[^5]:通称:春日神「天児屋命（あめのこやねのみこと）」が祝詞（のりと）を唱えた
[^6]:芸能の始祖神・日本最古の踊り子「天宇受賣命（あめのうずめのみこと）」が舞いを踊った
[^7]:顔を出したアマテラスを「天之手力男神（あめのたぢからおのかみ）」が引きずり出した

## ヤマタノオロチ（八俣遠呂智、八岐大蛇）
スサノオはたくさんの贖いを負い、天から地に追放され、出雲の鳥髪山[^8]に降った
[^8]:比定地は現在の鳥取県日南町と島根県奥出雲町の県境にある「船通山（せんつうざん）」

そこで、美しい娘を持つ老夫婦に出会い、8つの頭と尾を持つ巨大な怪物「八俣遠呂智（ヤマタノオロチ）」に、これまで7人の子を食われたこと、そして最後に残った末娘「櫛名田比売（クシナダヒメ）」が近々食べられてしまうことを知らされる

スサノオは、クシナダヒメを嫁にもらうことを条件に、強い酒と仕掛けを用いてヤマタノオロチを退治し、その身体の中から三種の神器の一つ「天叢雲剣（あめのむらくものつるぎ）[^9]」を手に入れ、それをアマテラスに贈った
[^9]:熱田神宮にある本体と、皇居にある形代の2つがある（草那藝之大刀（くさなぎのたち）とも言われる）
