# 天地開闢（てんちかいびゃく）
遥か遠い昔、卵の中身のような万物の根源の兆しが生まれた
そしてそれは長い年月をかけ、清らかな（陽の）気がたなびく「天」と、重く濁った（陰の）気を帯びた「地」に別れた
（ここまでは、漢籍『淮南子』『三五暦紀』を引用）

___

最初に現われた三柱の神「造化三神」は、何ら事績を語らず身を隠した
* 天之御中主神（アメノミナカヌシ） - 全ての信仰の根本、至高の神
* 高御産巣日神（タカミムスヒ） - 万物の生産・生成を司どる神
* 神産巣日神（カミムスヒ） - 活力の生産・生成を司どる神
（タカミムスヒは天皇守護のための神々の祖・カミムスヒは出雲系の神々の祖とされる）

続いて、天に浮かぶ油のように二柱の神が現われ、またすぐに身を隠した
* 宇摩志阿斯訶備比古遅神（ウマシアシカビヒコヂ） - 生命力・活力の神
* 天之常立神（アメノトコタチ） - 天の永久性を象徴する神

この五柱の神を、根元的な影響力を持つ特別な神「別天津神（ことあまつかみ）」と呼ぶ

***

別天津神に次いで、生まれた二柱の神は、別天津神と同様に単独で成った神「独神（ひとりがみ）」
* 国之常立神（クニノトコタチ） - 国土の形成・人類の生活の始原を表す
* 豊雲野神（トヨクモノ） - 天と地との間の虚空の空間を表す
（独身のまま子は得ず、すぐに身を隠し、以降表だって神話には登場しない）

その後に生まれた五代（男女二柱の対を一代と数える）の神々は、男女の形成を神格化した神々
（独神に対して、男女一対の神を「双神（ならびかみ）」ということもある）

| 宇比地邇神（ウヒヂニ）| 須比智邇神（スヒヂニ）|　ヒヂ・迩は「土・泥」を意味する
| 角杙神（ツヌグイ）| 活杙神（イクグイ）| 角は芽立ち・活は生命力を意味する
| 意富斗能地神（オオトノヂ）| 大斗乃弁神（オオトノベ）| 男女の身体の違い（心体の差）を意味する
| 於母陀流神（オモダル）| 阿夜訶志古泥神（アヤカシコネ）|　人体の完成（＝不足したところのない）を意味する
| 伊邪那岐神（イザナギ）| 伊邪那美神（イザナミ）| 「イザナ」は「誘ふ」の語幹（誘い合う男女神）

以上の七組十二柱を、神世七代（かみよななよ）と総称する
（または、「天」に別天津神が現れてから、「地」に島が創造される以前の時代を神世七代と呼ぶ）
