# 国生み/国産み（くにうみ）
別天津神たちは、神世七代（かみのよななよ）の最後に産まれた「伊邪那岐（イザナギ）・伊邪那美（イザナミ）」に、日本を完成させるよう命じ、「天沼矛（あめのぬぼこ）」を与えた

イザナギとイザナミは、天と地の間の虚空にある浮き橋「天浮橋（あめのうきはし）」に立ち、天沼矛で渾沌とした「地（大海）」をかき混ぜ、日本の国土の始まり「オノゴロ島[^1]」を創成した
[^1]:淡路島の属島の沼島・絵島、紀淡海峡の友ヶ島など複数の地が比定地とされている

イザナギとイザナミは「オノゴロ島」に降り、天地をつなぐ柱「天の御柱（みはしら）」と、御殿（家屋）「八尋殿（やひろどの）」を建て、それぞれの身体を見せ合い、イザナミの身体の足りない所（女性器）にイザナギの身体の余っている所（男性器）を差し塞んだ

一度目は女性であるイザナミのほうから誘ったため、まともな子供ではない「水蛭子（ヒルコ）[^2]」が生まれた
次に「淡島（あわしま）[^3]」が生まれたが、またしても不具の子だった
（イザナギとイザナミは水蛭子・淡島を葦船（あしぶね）に乗せ、流してしまう）

[^2]:始祖となった男女二柱の神の最初の子が生み損ないになるという神話は、東南アジアを中心とする「洪水型兄妹始祖神話」など界各地に見られる
[^3]:和歌山加太の淡嶋神社を総本社とする全国の淡島神社や淡路神社の祭神（神仏分離などにより現在は少彦名神等に置き変えられていることが多い）

イザナギとイザナミは別天津神へ相談したところ、女性から誘うのがよくないとされ、男性であるイザナギのほうから誘い結婚を誓い、日本の国土の始まり「大八島」、続いて「六島」を産んだ

4. 筑紫島（つくしのしま） - 九州
5. 伊伎島（いきのしま） - 壱岐島

7. 佐度島（さどのしま） - 佐渡島

[^4]:記紀には東北・関東地方がほとんど登場しない（北海道・沖縄と同じく、天皇家（ヤマト政権）とは異なる勢力が統治する地域だったと考えられる）
