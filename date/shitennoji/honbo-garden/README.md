四天王寺 🇯🇵 本坊庭園
Shitennoji Honbo Garden


拝観料 大人300円
拝観時間 8:30 ~ 16:30（4月~9月）または 16:00（10月~3月）


中国の説話「二河白道（にがびゃくどう）」に基づく日本庭園「極楽浄土の庭（ごくらくじょうどのにわ）」
（二河は生き地獄「火の河：怒り」と「水の河：欲深さ」を指し、その間にある「白道」を進むと極楽浄土に達し往生できるという教え）

入口付近の「釈迦の滝」から、三段滝石組「薬師の滝」がある「瑠璃光の池」へ流れる小川が「火の河」
畔に国登録有形文化財の水色い洋風建築「八角亭」が建つ「極楽の池」へと流れる小川が「水の河」
（例年8月中旬~9月中旬、極楽の池にはたくさんの蓮の花が咲く）

庭園入口から極楽の池まで、二つの小川（火の河・水の河）に挟まれた白砂の廻遊路「白道」を進んだ先には、極楽浄土の庭を見下ろす西側にある座視式庭園「補陀落の庭（ふだらくのにわ）」がある
（補陀落は、観音菩薩の降臨する霊場・観音菩薩が降り立つとされる伝説上の山を意味する）
