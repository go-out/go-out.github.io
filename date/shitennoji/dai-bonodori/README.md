毎年八月十一・十二日 午後六時半～八時半
Date(s) : August 11th, 12th every year 18:30 ~ 20:30

「聖徳太子千四百年御聖忌記念事業」として、2014年に復活してから毎年開催されている夏の恒例行事

例年、河内家菊水丸が国の重要文化財に指定される「石舞台」で音頭を務め、「亀の池」に特設された櫓の上で、社中がギター・和太鼓を演奏
踊り手は、亀の池を囲んで円陣を組み、手踊り・マメカチ・ナミハヤを踊る

ギター 石田雄一 
和太鼓 三条史郎

演奏される音頭は、全編八話の大作オリジナル河内音頭「聖徳太子一代記」（2014年制作開始・2023年完結）の一部
