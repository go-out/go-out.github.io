四天王寺 🇯🇵 Shitennouji

住所：大阪市天王寺区四天王寺１丁目１１−１８
1-11-18 Shitennoji Tennoji-ku Osaka, Japan


年間行事 主な祭事

毎年一月十四日 「どやどや」
梁の上から牛王宝印（ごおうほういん）という護符が投入れられ、高校生がふんどし・はちまき姿で奪い合う

毎年四月二十二日 「聖霊会の舞楽」
聖徳太子の御忌にその聖霊をまつる法会（国の重要無形民俗文化財）

毎年春分・秋分の日と前後三日間 「彼岸会」
中日（春分・秋分の日）午後五時二十分から、極楽門前で、沈む夕陽を拝む行事「日想観」が行われる

毎年八月十一・十二日 「四天王寺盆踊り大会」
午後六時半から 於：石舞台と亀の池周辺

毎月二十一・二十二日 「四天王寺骨董市」
午前八時半から午後四時頃まで 関西最大級の骨董市が毎月開催される
