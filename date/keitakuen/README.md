慶沢園 🇯🇵 Keitakuen

天王寺公園内にある日本庭園（林泉式回遊庭園） 大阪市指定文化財

1918年（大正7年） 住友家の茶臼山本邸庭園として完成
1925年（大正14年） 旧本邸敷地（現：大阪市立美術館）・茶臼山とともに大阪市に寄贈
1936年（昭和11年） 大阪市立美術館とともに一般公開

作庭：小川治兵衛／設計：木津聿斎 


入園料 大人150円

開園時間 9:30~17:00
（5・9月の土・日・祝、7・8月の金・土・日・祝は 20:00まで）

閉園日 月曜日（月曜が祝祭日の場合は翌日閉園）
※ リニューアル工事のため、令和6年4月1日（月曜日）から令和7年春（予定）まで長期休園
