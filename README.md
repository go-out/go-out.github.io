* Road (Rd.): 道・道路・鉄道・街道
* Way: roadから外れた小さな脇道。

* Street (St.): 街路・通り・…街・（歩道と区別した）車道
* Avenue (Ave.): 並木道・（都市の）大街路・大通り
* Boulevard (Blvd.): 広い並木街路・大通り（通常は真ん中に中央分離帯が通っている）
* Lane (Ln.): 都市部でよくみられる狭い道（基本的にはBoulevardの反義語）

* Drive (Dr.): 車道・まわりの環境にあわせて作られた、長く曲がった道（湖や山にある道など）
* Path: （人に踏まれてできた）小道・細道・（公園内の）歩道・通り道・通路
* Sidewalk: （舗装された）側道・歩道
