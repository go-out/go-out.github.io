「春分の日」を中日に、前後3日間をあわせた7日
冬の寒さ（余寒）が和らぎ、「自然のあらゆる生命が若々しく盛り上がる時」

---

春分の日は、「自然をたたえ、生物をいつくしむ」日本の国民の祝日
（昼と夜の長さが等しくなる日といわれますが、実際は昼の方が少し長い）

具体的な月日は定められておらず、国立天文台が毎年2月に公表する暦要項により、翌年の春分の日と秋分の日の日にちが確定します。

（例年3月20日～21日ごろ）

日本では、桜・梅・桃など、春の花が咲き始める

桜
開花時期は、3月中旬〜5月上旬
都島区　(サクラ・コスモス)
此花区　(サクラ・チューリップ)

梅
開花時期は、1月下旬〜4月下旬
（品種によっては冬から開花する「早春」の花）

桃
開花時期は、3月中旬〜4月下旬
（桃から桜の順番で開花することが多い）
天王寺区　(モモ・パンジー)
阿倍野区　(モモ・ペチュニア)


杏
開花時期は、3月〜4月

白木蓮（ハクモクレン）
3月中旬から4月上旬 コブシよりやや早く咲く
ハクモクレンは花びら（に見えるもの）が9枚あるのに比べ、コブシは6枚

紫木蓮（シモクレン）

花言葉は「自然への愛」「荘厳」「恩恵」

城東区　(モクレン・コスモス)
