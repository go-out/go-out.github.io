大阪市南西部では、河川や運河を行き来する市営の渡船が運航しています。

運航時間は、概ね午前6時台から午後9時台。運賃は無料。歩行者および自転車のみ乗船できます。
古来数多くの川が流れ、水の都と呼ばれた大阪には、人々の往来のために架橋の代替（または補完）として運航する渡船が各所にありました。
（大正9年4月、旧道路法の施行により渡船は無料となり、昭和7年4月以降、渡船のほとんどは市の直営方式となりました。）

都市施設が整備された今、多くの渡船は廃止されたが、現在でも8か所の渡船場（15隻の船）が、地域の人々に利用されていて、繁華街や観光地では見られない大阪に触れることができるスポットとしても、静かな人気を呼んでいます。
昭和10年頃には渡船場31ヶ所、保有船舶数69隻（機械船32隻、手漕ぎ船37隻）、年間利用者は歩行者が約5752万人、自転車等が約1442万台とピークに達しました。
その後、渡船の利用は次第に減少し、平成元年（1989年）に現在の渡船場の数（8ヶ所）となり、利用者数は約152万人にまで減少しました。
